    DhravyaAPI

    A REST api to get the text from an image (OCR), get random memes from reddit based on the provided topic, Make a *fancy* QR code, or just get compliments, jokes, or would you rather questions

  • 2021 - Present

    Spacebot

    Spacebot, a discord bot, is meant to replace every mainstream bot out there to reduce clutter, engage communities and bring people together!. Modules and libraries used: Discord.py, various API wrappers, Pillow, Beautiful Soup, ffmpy
